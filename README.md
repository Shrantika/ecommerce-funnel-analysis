# E-commerce Funnel Drop-off Analysis

## Objective
Analyze user behavior across an e-commerce funnel (view → cart → purchase) to identify drop-off points and improve conversion rates.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL (SQLite)
- Jupyter Notebook (VS Code)

## Key Insights
- Only ~24.77% of users move from view to cart
- Only ~19.42% proceed from cart to purchase
- Several products show extremely low conversion rates (1–5%)

## Recommendations
- Improve product pages (images, descriptions)
- Add reviews and ratings
- Optimize pricing and discounts
- Simplify checkout experience
